Solapex is an open-source tool designed to automate the calculation of available "free area" on rooftops using satellite or aerial imagery. By identifying obstructions (chimneys, water tanks, AC units) and calculating usable surface area, Solapex helps users determine the maximum number of solar panels a roof can accommodate.
